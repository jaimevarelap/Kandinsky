# Kandinsky

Red neuronal profunda entrenada para crear cuadros al estilo de Kandinsky a partir de trazos sencillos hechos por el usuario.
- Conditional GAN
- UNet (Basada en la arquitectura de pix2pix
- Image to image translation

Controles:
- C : Cícrulo
- K : Curva libre
- L : Línea
- P : Escoger color
- F : Llenar
